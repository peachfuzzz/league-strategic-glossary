id: skill-test
term: Skill Test
tags: [abstract-concepts]
---

A task, encountered regularly during games, whose outcome affects the game state. Skill tests are the abstract categories of problems that League of Legends asks of a player that wants to win. Skill tests can be categorized broadly and narrowly. For example, positioning is a skill test asked of all champions, while pet micromanagement only concerns champions that summon controllable pets.
